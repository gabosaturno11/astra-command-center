# THE BLOCK ARCHITECTURE

Every piece of content is assembled from modular blocks.

```
H  = HOOK      →  Attention capture
B  = BUILD     →  Context expansion
T  = TURN      →  Pivot/Reframe
P  = PEAK      →  Maximum intensity
L  = LAND      →  Command/Invitation
C  = CLOSE     →  Final resonance
```

## Block Formulas by Scale

| Formula | Structure | Scales | Description |
|---------|-----------|--------|-------------|
| MICRO | H → L | 1-2 | Hook straight to landing |
| STANDARD | H → B → T → L → C | 3-4 | Full short-form arc |
| EXPANDED | H → B → B → T → P → L → C | 5 | Multiple builds, peak moment |
| EPIC | H → B → T → B → T → P → B → L → C | 6-7 | Multiple cycles, callbacks |

## The Fractal Rule

At any scale, a block can contain its own H-B-T-P-L-C structure at the scale below.

A Scale 5 BUILD block might be a complete Scale 4 piece with its own hook, build, turn, land, close.

**This is what makes the system infinitely nestable.**
