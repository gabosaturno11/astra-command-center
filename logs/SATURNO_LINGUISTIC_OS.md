# PART 8: PRACTICAL APPLICATION

## Example: Complete Workflow

**INPUT:** Voice memo: "I've been thinking about how we resist the very things that would help us..."

### STEP 1: Capture
Raw input logged. No editing.

### STEP 2: Theme
- Core theme: Resistance to help
- Polarity: Resistance ↔ Surrender
- Tension: Why we fight what we need

### STEP 3: Scale Selection
- Voice memo depth suggests: ORGAN level (Section/Email, 150-500 words)
- Could extract: MOLECULE quotes, CELL caption

### STEP 4: Mode Assignment
- Primary: /PhilosopherMode (universal observation)
- Secondary: /CompanionMode (shared struggle)
- Close: /TeacherMode (practical landing)

### STEP 5: Linguistic Layer
- **Structural:** Use passive voice initially (agent suppression for universal feel)
- **Prosodic:** Start long (contemplative), end short (punch)
- **Semantic:** Frame as "pattern" not "problem"
- **Pragmatic:** Use presupposition ("We all do this...")
- **Cohesion:** Build "this" → "that" → "it" chain
- **Register:** Warm formal

### STEP 6: Block Formula
H (Hook observation) → B (Build the pattern) → T (Turn to why) → L (Land with invitation) → C (Close with hope)

### STEP 7: Generate

**OUTPUT (ORGAN Scale, ~200 words):**

> We resist the very things that would help us.
>
> Not consciously. Not with intent. It happens in the body before it reaches the mind—the subtle turning away from the stretch that would release us, the avoidance of the conversation that would free us, the infinite delay of the work that would fulfill us.
>
> This isn't weakness. It's protection. 
>
> Something ancient in us learned that change equals danger. That vulnerability equals death. So we built walls against our own medicine and called it discipline, called it caution, called it "not the right time."
>
> But here's what I'm learning: the resistance itself is data. It points directly at the threshold we most need to cross.
>
> Next time you notice yourself turning away—from the hard conversation, the unfamiliar movement, the uncomfortable truth—pause. Don't push through. Just... notice.
>
> That's where the door is.
>
> With Love,
> Gabo
