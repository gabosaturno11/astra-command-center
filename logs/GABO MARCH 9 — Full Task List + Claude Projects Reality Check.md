### Claude Projects Platform Limits (March 2026)
Understanding these limits is critical before building new projects:

- **File size**: 30MB per file[^2]
- **Number of files**: Unlimited, but total extracted text must fit within ~200,000 tokens[^3]
- **Knowledge usefulness**: Community reports effectiveness drops below ~25-30% capacity — Claude stops pulling from KB reliably after that[^4]
- **Context window**: 200K tokens (Claude 3.5 Sonnet / Claude 4)[^3]
- **Caching**: KB files are cached — repeat queries don't re-count against limits[^5]
- **RAG mode**: Available for expanded KB capacity beyond context window[^5]
- **Cross-chat**: Context is NOT shared across chats within a project unless added to KB[^6]
### Perplexity Spaces Platform Limits (March 2026 — Max Plan)
- **Files per Space**: 50 files (Pro), 500 files (Enterprise)[^7]
- **File size**: 40-50MB per file depending on plan[^8][^7]
- **Retention**: 90 days on Pro[^8]
- **Models**: Can toggle between Claude, GPT-4o, Sonar, and others within Spaces[^9]
- **Web search**: Built-in — Spaces can pull real-time data, Claude Projects cannot[^10]
### The Decision: Claude Projects vs Perplexity Spaces
| Dimension | Claude Projects | Perplexity Spaces |
|-----------|----------------|-------------------|
| Deep work (coding, long-form) | ✅ Superior — 200K context, deep reasoning[^10] | ❌ Not built for this |
| Research with citations | ❌ No web search | ✅ Superior — real-time web + citations[^10] |
| Knowledge base reliability | ⚠️ Degrades at ~30% capacity[^4] | ✅ RAG with source attribution |
| Cross-session memory | ❌ Only via KB files[^6] | ✅ Conversation history within Space |
| Cost | $20/mo (Pro) or $100/mo (Max) | Included in Max ($200/mo) |
| Custom instructions | ✅ Detailed per-project[^11] | ✅ Similar capability[^12] |
| Model choice | Claude only | Multiple models per Space[^9] |

**Bottom line**: Use BOTH. Claude for execution (coding, long-form writing, PDF generation). Perplexity for research, routing, and real-time intelligence.[^10]
